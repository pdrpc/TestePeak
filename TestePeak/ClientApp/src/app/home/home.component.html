Olá, Peakers!

Esta é a minha aplicação single page de teste para Peak Invest, construído com:

  • ASP.NET Core e C# para código server-side multiplataforma
  • Angular e TypeScript para código client-side
  • Bootstrap para fazer o layout e os estilos

O que este projeto faz?

  • Consulta:.Traz um nome de uma lista da Web-API segundo um indice fornecido pelo usuário. Isso acontece através de um requisição GET.
  • Parcelas. Manda os valores e a quantidade de parcelas e juros para API através de uma requisição POST. A API faz o calculo do total a ser pago incluindo os juros

Muito obrigado pela oportunidade, por enquanto é isso e nos vemos em breve!
